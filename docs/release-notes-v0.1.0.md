# ElectrochemTools v0.1.0

First experimental release with unified EIS cleaning, batch direct-Debye/piecewise-linear DRT, matched MATLAB/DRTtools golden validation, strict resumable computation manifests, and `R_QR` fitting.

This is not a full DRTtools replacement. It does not provide RBF DRT, HMC, Bayesian HT, additional ECM models, a GUI, or Python bindings. The prebuilt archive is for 64-bit Windows MSVC only.
